# Control de Prácticas Profesionales y Servicio Social

Aplicación móvil Android desarrollada en Kotlin para facilitar el control y seguimiento de horas de prácticas profesionales y servicio social.

# Objetivo

Permitir a los estudiantes registrar actividades, consultar su historial de registros y visualizar el avance de horas acumuladas de manera sencilla y accesible.

# Tecnologías

* Kotlin
* Android Studio
* Jetpack Compose
* Room Database
* GitHub

# Funcionalidades previstas

* Registro de actividades diarias
* Control de horas acumuladas
* Historial de actividades
* Edición y eliminación de registros
* Visualización de progreso
